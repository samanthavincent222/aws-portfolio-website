# AWS Portfolio Website

## Overview
This project is a personal portfolio website created using HTML, CSS, and JavaScript and deployed on Amazon S3 using static website hosting. The purpose of this project is to build a professional online presence and gain hands-on experience with AWS cloud services.

## Technologies Used
- HTML
- CSS
- JavaScript

## AWS Services Used
- Amazon S3 (Static Website Hosting)
- AWS IAM (Access Management)

## Deployment Process
1. Created an S3 bucket and configured it for static website hosting
2. Uploaded portfolio website files to the S3 bucket
3. Configured bucket policy to allow public access
4. Accessed the website using the S3 website endpoint
